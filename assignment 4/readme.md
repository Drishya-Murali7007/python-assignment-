# Weather Data Visualizer

This project loads real weather data from a CSV file, cleans it, performs statistical analysis, and generates multiple visualizations including temperature trends, rainfall totals, and humidity relationships.

## Features
- Load and clean CSV data
- Handle missing values (NaN)
- Convert dates to datetime format
- NumPy-based statistical calculations
- Matplotlib visualizations (line, bar, scatter)
- Monthly weather aggregation using groupby
- Exports cleaned dataset and summary

## Files Included
- weather.csv (raw dataset)
- cleaned_weather_data.csv (processed dataset)
- PNG plots
- summary.txt
- main.py (complete analysis code)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
